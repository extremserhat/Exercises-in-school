Unterschied von Klassen und ID-Selektoren

(#) ID-Selektor

Selektiert fenau ein Element mit der gegebenen ID. Die ID-Bezeichnung darf nur einmal im Dokument vorkommen.

(.) Class-Selektor

Selektiert alle Elemente denen angegebenen KLasse zugewiesen wurde. Ein Element kann mehrere Klassen Besitzen.
